# **App Name**: CarpoolConnect

## Core Features:

- Authentication & Profile: Secure login with company email (SSO or OTP) and employee profile management.
- Ride Creation (Driver): Drivers can post rides with start/end locations, time, available seats, and optional pickup points.
- Ride Discovery (Passenger): Passengers can find rides based on route, destination, and time; request to join available rides.
- In-App Communication: Drivers and passengers can communicate via in-app messaging; real-time ride status updates and notifications.
- Real-time Mapping & Navigation: Integration with Google Maps for live route display, driver path, and pickup points.
- Carpool Matching Assistant: An AI-powered tool assists in matching drivers and passengers based on location, schedule, and preferences, ensuring optimal carpool opportunities.
- Eco-Friendly Leaderboard: Tracks and displays top eco-friendly commuters within the organization based on rides shared and distance covered.

## Style Guidelines:

- Primary color: Deep teal (#008080), reflecting trustworthiness, and safety in transportation.
- Background color: Light teal (#E0F8F8), offering a calming and clean base.
- Accent color: Vibrant coral (#FF7F50) to highlight CTAs and important notifications.
- Body and headline font: 'Inter' (sans-serif) provides a modern and readable experience for all text.
- Use a set of clean, modern icons that relate to transportation and environmental sustainability.
- A tab-based layout with clear distinctions between 'Start Ride' and 'Join Ride' functionalities for intuitive user navigation.
- Subtle animations to acknowledge user interactions, enhancing the overall app experience with smooth transitions.
